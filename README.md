# Mysql_project
Project Overview:-
This project focuses on analyzing the Global Supermarket Sales Dataset using MySQL.
The objective is to extract meaningful business insights such as sales trends, profit analysis, customer behavior, and regional performance using SQL queries.

Analyze total sales and profit performance
Identify top-selling products and categories
Find high-performing regions and markets
Analyze monthly and yearly sales trends
Identify loss-making products
Customer purchase behavior analysis

Business Insights:-

Identified top revenue-generating products
Found regions contributing highest profit
Detected products causing losses
Observed seasonal sales trends
Analyzed impact of discounts on profitability
